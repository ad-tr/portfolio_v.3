# Portfolio Adrien Tranchant

Portfolio personnel moderne et créatif pour présenter mes projets en intelligence artificielle et deep learning.

## Technologies utilisées

- **Next.js 15** - Framework React pour les applications web
- **TypeScript** - JavaScript typé pour plus de robustesse
- **TailwindCSS** - Framework CSS utility-first
- **Framer Motion** - Bibliothèque d'animations fluides
- **React 19** - Bibliothèque UI

## Fonctionnalités

- Design moderne et minimaliste avec une touche futuriste
- Animations fluides avec Framer Motion
- Design 100% responsive (mobile, tablette, desktop)
- Navigation smooth scroll entre les sections
- Page 404 personnalisée
- Optimisé pour les performances

## Installation

```bash
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev

# Builder pour la production
npm run build

# Lancer en production
npm start
```

## Développement

Le site sera accessible sur [http://localhost:3000](http://localhost:3000) en mode développement.
